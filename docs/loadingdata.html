Loading Data into Hyscorean

Files containing HYSCORE data can be loaded into the program via the Load button:

By pushing the button an OS window will appear requesting the user to select the files to be loaded. Multiple files can be loaded by selecting all of them in the window.

Hyscorean is compatible with the following file extensions:

    .DSC .DTA   Bruker BES3T data file format used on ELEXSYS and EMX spectrometers
    .mat     MATLAB output file format used in the EPR@ETH AWG spectrometer
    .txt     Text files in ASCII format

NOTE:
Loading new data will reset Hyscorean deleting any data stored into the variables as well as all displays.

When loading BES3T data files (either .DSC or .DTA) it is important that the complementary .DTA or .DSC (and in some cases the .XGF) files are present in the same folder as the loaded file. Otherwise, the data will not be loaded correctly leading to an error.

Once the file(s) have been loaded, the indicator below the Load button will display how may files were loaded. The user can control at any moment which files are currently loaded into the program by pressing the (…) button next to the display. This will open a window with a list of all loaded files where the user can remove loaded files selectively:
