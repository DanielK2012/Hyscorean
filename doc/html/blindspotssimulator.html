Introduces a τ-dependent blind spot behavior of the HYSCORE spectrum. These blind spots can be highly damaging and dangerous for the interpretation of HYSCORE spectra. This is a reason why HYSCORE experiments should be recorded at different τ-values. In Hyscorean, the blind spots corresponding to the currently selected τ-values in the Tau-Selection list box can be simulated via

where N is the number of combined τ-values. This blind spot map can be superimposed onto the displayed spectrum to see which parts of the spectrum may be suppressed by such blind spots. This can be enabled and disabled from the Blind Spots button. The blind spots are identified as the darker spots in the superimposed map, whereas the brighter spots represent the zones of maximal intensity.

There is also the possibility to simulate the blind spots from a constructed sequence of τ-values. This is helpful to interactively find appropriate τ-values for an experiment. The simulator can be opened by pressing the Simulator button.

The current τ-values are given in the top of the figure. The current τ-value can be changed by moving the slider. Once a desired τ-value is found it can be added to the list by pressing the Add Tau button, which will fix the τ-value and add a next one to be changed:

The current experimental spectrum can be overlaid upon the simulated blind spots via the Overlap experimental spectrum checkbox.

NOTE:
The slider can adopt τ-values between 100ns and 350ns. Also the response time for the simulator can be longer than expected for computers with low graphics capabilities.
