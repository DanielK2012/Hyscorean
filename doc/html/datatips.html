Exploring the Data with Datatips

HYSCORE spectra contain different kinds of information encoded, e.g. in the frequencies of the crosspeaks and the distances between pairs of crosspeaks. In order to provide a smooth and convinient exploration of the data, Hyscorean includes a custom datatip feature. This feature can be accessed once a spectrum has been displayed and the user can start it at any time by pressing the [x;y] button next to the zoom-in/out buttons in the main display. Once the button is pressed the datatips will be shown as follows:

The datatip consists of a set of two points: one at the pointer position and another at its reflection about the (anti)-diagonal and a red line crossing both points. The datatip itself displays the frequencies corresponding to the pointer position. Additionally the distance in frequency between the two displayed points is given in the datatip as Δ. To stop the datatip feature, the [x;y] must be pressed again.

NOTE:
This feature is incompatible with the zoom-in/out features and may lead to graphical bugs. Additionally, all further function evaluation is stopped until this feature is stopped. Therefore, the user needs to stop this via the [x;y] button.
