Saving the Validation Results

Once the validation has been completed and the user is satisfied with the results, the results can be exported from the module in two ways. A first way involves simply detaching the validation display to a new window (via the corresponding button in the right-top corner of the display) from where the user can save it via MATLAB.

All relevant data and information can easily be saved via the Save Validation button. Once the button is pressed an OS window will prompt for the user to select a SaveName and folder where the files are to be created. Then, similar as in the saving protocol for the main Hyscorean GUI, a sequence of files will be created:

Filename: Date_SaveName_...

  • ...UpperBound.fig

    A MATLAB figure files containing an exact copy of the upper bound results as in the validation module display.

  • ...LowerBound.fig

    A MATLAB figure files containing an exact copy of the lower bound results as in the validation module display.

  • ...Uncertainty.fig

    A MATLAB figure files containing an exact copy of the uncertainty results as in the validation module display.

  • ...ValidationReport.pdf

    An automatically generated report containing a summary of the validation parameters employed as well as a collection of the different displays created by the validation.

  • ...ValidationData.mat

    A MATLAB data file containing a set of data necessary by the EasySpin Fitting module as input (see here). 

    .Uncertainty 			% Spectral uncertainty data
.MeanSpectrum 			% Spectral mean data
.LowerBound			% Spectral lower bound data
.UpperBound			% Spectral upper bound data
.ValidationSpectra  		% Set of spectra obtained during validation
.ValidationParameters		% Parameter vectors employed for validation
.ParameterSets			% Parameter sets of corresponding spectra
.FrequencyAxis1			% X-axis of frequency-domain signal
.FrequencyAxis2			% Y-axis of frequency-domain signal

NOTE:
If the numbers of trials is too large, the Date_SaveName_ValidationData.mat file will consume a considerable amount of memory. Should this exceed 1GB of memory the program will ask the user whether the file is to be saved. The rest of the files will be created as usual.
