Uninstalling Hyscorean

To remove Hyscorean from the computer and from MATLAB the user must follow these instructions:

    1) Remove Hyscorean from the MATLAB search path. This can be done via the MATLAB GUI selecting the Home>Environment>Set Path option and then in the new window selecting all Hyscorean paths and removing them via the Remove button. Make sure to exist the window via the Save button to ensure the paths are removed in further session.

    2) Remove the MATLAB preferences set by Hyscorean by executing the following command in the MATLAB console. 

    rmpref('hyscorean')

    3) Remove all files from the computer by deleting them from the location chosen by the user during installation.

At this point Hyscorean will be completely removed from the system as prior to installation. Should the user want to re-install Hyscorean, the installation will have to be done as described at the start of this manual.
