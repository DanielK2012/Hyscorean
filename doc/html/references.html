If you publish results obtained with Hyscorean, please cite

Luis Fabregas Ibanez et al.
Non-uniform sampled HYSCORE spectra: measurement, processing and analysis with Hyscorean
J. Magn. Reson. 178(1), 42-55 (2019)

If you publish results obtained with the Hyscorean fitting module, please cite

Stefan Stoll, Arthur Schweiger
EasySpin, a comprehensive software package for spectral simulation and analysis in EPR
J. Magn. Reson. 178(1), 42-55 (2006)

EasySpin's pulse simulation capabilities are described in detail in

Stefan Stoll, R. David Britt
General and efficient simulation of pulse EPR spectra
Phys. Chem. Chem. Phys. 11, 6614-6625 (2009)

A nice review on non-uniform sampling applied to magnetic resonance can be found in

Jeffrey C. Hoch et al.
Nonuniform Sampling in Multidimensional NMR
eMagRes 181-195, (2012)
